# 云枢管理平台系统介绍

## 一、系统概述
云枢管理平台是面向广告代理行业的CRM管理系统，解决多平台账户管理、资金流转、客户关系管理等核心业务问题。
**核心价值**：统一管理多平台账户、模板化业务流程、实时资金监控、数据驱动决策

## 二、涉及部门与核心功能
### 财务部门
1. **项目资金池管理**
   - 按项目管理资金，授信额度、余额管理
   - 对公/对私资金分类管理
   - 资金分配和调整
2. **资金流转管理（充转退）**
   - 充值：对公/对私充值、钱包充值、子端口充值
   - 退款：退款申请、审批、执行
   - 转账：账户间转账、批量转账
   - 充转退记录查询统计
3. **客户到账管理**
   - Excel模板批量导入
   - 自动解析到账信息（项目、金额、时间、资金性质等）
   - 到账记录与资金池关联
4. **银行账户管理**
   - 统一管理银行账户
   - 关联客户和项目
### 业务部门
1. **客户管理**
   - 客户档案（主体、商务、运营方式、合作渠道）
   - 客户分类（自运营、走量、收量）
   - 客户状态跟踪、数据统计
2. **项目管理**
   - 项目全生命周期管理
   - 项目与客户、渠道、资金池关联
   - 项目数据统计
3. **账户主体管理**
   - 多平台统一管理：快手（磁力引擎、磁力金牛、大健康、磁力聚星）、巨量（巨量广告、巨量千川、巨量本地推、巨量星图）、腾讯（广点通）
   - 账户主体与项目、渠道关联
4. **广告主信息管理**
   - 广告主账户信息、余额查询
   - 广告主与项目关联
5. **渠道管理**
   - 多渠道统一管理
   - 渠道返点配置
### 运营部门
1. **素材管理**
   - 素材库管理、预审状态跟踪
   - 批量前测功能
2. **返点管理**
   - 返点模板（公返、私返比例）
   - 媒体返点配置和统计
3. **广告ID管理**
   - 广告账户ID管理、绑定变更
   - 账户余额查询、状态跟踪
4. **数据分析**
   - 消耗数据分析、回款监测
   - 数据对比分析
### 系统管理
1. **用户与权限管理**
   - 用户账号、角色权限、部门管理
   - 基于角色的菜单和数据权限控制
2. **系统配置**
   - 系统参数、字典数据管理
3. **系统监控**
   - 操作日志、性能监控、异常告警
   
## 三、预计功能（规划中）
### 【预计】数据隔离
- 部门数据隔离、项目数据隔离、基于角色的权限隔离
### 【预计】银行自动获取流水
- 银行接口对接、自动获取流水、自动识别到账信息、资金池自动更新
### 【预计】资金池到账退款自动处理
- 基于银行流水自动处理到账和退款
### 【预计】电子税务客户自助开票
- 电子税务系统对接、客户自助开票申请、开票信息自动同步
### 【预计】根据开票回款监测
- 自动监测回款情况、回款预警、回款统计分析
